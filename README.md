## Product
**라즈베리파이를 이용한 AI 매장 분석 카메라**  
* 오프라인 매장의 정보(방문객 수, 관심영역) 제공
* 해당 정보를 바탕으로 데이터 분석
* 분석된 데이터를 통한 맞춤 마케팅 전략 제공
